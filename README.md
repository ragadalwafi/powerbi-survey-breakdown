# Data Professional Survey Breakdown
This project is an interactive Power BI dashboard that analyzes survey data from data professionals. It provides insights into salaries, career paths, job satisfaction, and industry trends to better understand the data field.

---
## Objectives
- Analyze salary distribution across different data roles
- Understand work-life balance and salary satisfaction
- Explore career transitions into data
- Identify popular programming languages
- Highlight what professionals value most in their careers

## Key Insights
- Data Scientists have the highest average salary among surveyed roles
- Work-life balance satisfaction is higher than salary satisfaction
- A large portion of professionals transitioned into data careers
- Python is the most popular programming language
- The top career priority is better salary, followed by remote work

---
## Dashboard Features
- Average Salary by Job Title (Bar Chart)
- Happiness with Work-Life Balance & Salary (Gauge Charts)
- Favorite Programming Language by Role (Stacked Bar Chart)
- Survey Demographics (Age, Country Distribution)
- Education Level Breakdown
- Career Switch Analysis
- Career Priorities Visualization
---
## Filters
**Country-based** filtering to explore insights for specific regions.
